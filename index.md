# Feeling Palette 개인정보처리방침

**시행일자**: 2026년 4월 18일
**최종 수정일**: 2026년 4월 18일

본 개인정보처리방침("방침")은 Feeling Palette 애플리케이션("앱")이 이용자의 개인정보를 어떻게 수집·이용·보관·보호하는지 설명합니다. 이용자는 앱을 이용하기 전에 본 방침을 주의 깊게 읽어주시기 바랍니다.

---

## 1. 수집하는 정보 및 수집 방법

### 1.1 이용자가 직접 입력하는 정보
- **일기 내용**: 이용자가 앱에 작성하는 감정일기 본문
- **앱 잠금 비밀번호(PIN)**: 이용자가 설정한 4자리 비밀번호 (해시 처리되어 저장됨)

### 1.2 자동으로 수집되는 정보
- **기기 정보**: 운영체제 버전, 기기 모델, 화면 해상도, 언어 설정
- **광고 식별자**: Android 광고 ID(ADID), iOS 광고 식별자(IDFA) — 이용자가 운영체제 설정에서 거부 가능
- **앱 사용 통계**: 기능 이용 빈도 (광고 SDK가 측정)

### 1.3 선택적으로 수집되는 정보
- **Google 계정 정보**: Google Drive 백업 기능을 사용하는 경우, Google 계정 이메일 주소 및 인증 토큰
- **생체 인증 정보**: 앱 잠금 해제에 지문/Face ID 사용 선택 시 (인증 결과만 앱이 받으며, 생체 정보 자체는 운영체제가 관리하고 앱은 접근할 수 없음)

---

## 2. 수집한 정보의 이용 목적

| 수집 정보 | 이용 목적 |
|-----------|-----------|
| 일기 내용 | AI 감정 분석 서비스 제공, 이용자 본인의 열람·조회 |
| PIN 해시 | 앱 접근 제어 (본인 확인) |
| 기기 정보·광고 식별자 | 광고 게재 및 성과 측정, 서비스 품질 개선 |
| Google 계정 정보 | 이용자 본인의 Google Drive에 일기 백업·복원 (선택 기능) |

---

## 3. 정보의 저장 위치 및 보관 기간

### 3.1 로컬 저장 (이용자 기기 내부)
- **일기 내용**: 이용자 기기의 SQLite 데이터베이스에만 저장됩니다. 개발자의 서버로 전송·저장되지 않습니다.
- **PIN 해시 / 생체 인증 설정**: 기기의 보안 저장소(Android Keystore / iOS Keychain)에 암호화 저장됩니다.
- **보관 기간**: 이용자가 앱을 삭제하거나 앱 내 "데이터 초기화" 기능을 사용하기 전까지 유지됩니다.

### 3.2 서버 처리 (AI 분석)
- 이용자가 "AI 감정 분석" 기능을 실행하면, 해당 일기 본문이 개발자의 분석 서버(`feeling-api-aws.sedoli.co.kr`)로 전송됩니다.
- 서버는 분석 결과를 즉시 반환한 뒤 요청 내용을 저장하지 않습니다. (처리 과정에서 발생하는 일시적 로그는 최대 30일 보관 후 자동 삭제됩니다.)

### 3.3 Google Drive 백업 (선택)
- 이용자가 백업을 실행하면 일기 데이터가 JSON 파일로 생성되어 이용자 본인의 Google Drive "앱 전용 폴더"에 업로드됩니다.
- 이 백업 파일은 이용자 본인만 접근·삭제할 수 있으며, 개발자는 접근 권한이 없습니다.

### 3.4 광고 데이터
- 광고 식별자 및 관련 데이터는 광고 네트워크(Google AdMob)의 정책에 따라 보관됩니다.

## 4. 제3자 서비스 및 정보 공유

본 앱은 다음 제3자 서비스를 이용하며, 해당 서비스의 개인정보 처리방침이 함께 적용됩니다.

### 4.1 Google AdMob (광고)
- 수집 범위: 광고 식별자, 기기 정보, 대략적 위치(국가 단위), 광고 상호작용 정보
- 목적: 맞춤형 광고 게재 및 광고 성과 측정
- 정책: <https://policies.google.com/privacy>
- 이용자는 기기 설정에서 "광고 추적 제한" 또는 ATT(iOS) 거부 선택 시 비맞춤형 광고만 수신합니다.

### 4.2 Google Drive API (선택 기능)
- 수집 범위: Google 계정 이메일, 인증 토큰, Drive 파일 접근 권한
- 목적: 이용자 본인의 Google Drive에 일기 백업/복원
- 정책: <https://policies.google.com/privacy>
- 이용자는 언제든 Google 계정 설정에서 앱 권한을 해제할 수 있습니다.

### 4.3 AI 분석 서버 (개발자 운영)
- 운영자: Feeling Palette 개발자
- 수집 범위: AI 분석을 위해 전송된 일기 본문 (분석 후 미저장)
- 목적: 감정 분석 결과 반환
- 접근 제한: HTTPS 통신으로 전송되며 개발자만 접근 가능

본 앱은 위에 명시된 경우 외에 이용자의 개인정보를 제3자에게 판매·양도·공유하지 않습니다.

## 5. 이용자의 권리

이용자는 다음 권리를 행사할 수 있습니다.

1. **열람·수정**: 앱 내에서 작성한 일기를 언제든 열람·수정 가능
2. **삭제**: 개별 일기 삭제 또는 앱 내 "데이터 초기화" 기능으로 전체 삭제 가능
3. **광고 추적 거부**: 기기 OS 설정에서 광고 식별자 재설정 또는 추적 제한 활성화
4. **Google 계정 권한 회수**: Google 계정 설정 → 타사 앱에서 Feeling Palette 권한 제거
5. **앱 삭제**: 앱을 삭제하면 기기 내 저장된 모든 데이터가 삭제됨

## 6. 보안 조치

- 앱 잠금 기능을 통한 접근 제어 (4자리 PIN + 선택적 생체 인증)
- PIN은 SHA-256 해시 + 솔트 + 스트레칭으로 저장 (원본 복구 불가)
- 기기 보안 저장소(Keystore/Keychain) 활용
- 서버 통신은 TLS(HTTPS) 암호화
- 개인정보에 접근할 수 있는 인력은 개발자 본인으로 제한
